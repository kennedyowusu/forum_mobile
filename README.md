# ğŸ“– [Metric WebApp] <a name="about-project"></a>

**[Evolve]** is a react based responsive web application that allows users to view countries around. The web app has two pages: the home page and the details page. The home page displays a list of countries that could be filtered by country name. When the user clicks (or taps) on a country, the application navigates to the details page.
